## How to run the frontend

```sh
streamlit run frontend/main.py
```

## How to run backend

```sh
cd backend
pip install -r requirements.txt
cd ..
python backend/get_transcript.py
```

`chat.py` and `rag.py` are still in-progress, so running them will cause an error
